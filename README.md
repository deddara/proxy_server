# Simple proxy server, especially for mysql

## **`-help`**  -  to see usage


Flag         | Description  | Default value
------------ | -------------| -------------
-ip | ip address to connect | 127.0.0.1
-p | port to connect| 3306
-sp | server port to run | 3030
-f | file name for log| 'mysql.log'
-t | thread numbers| 5
